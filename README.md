# Google-Chrome-prawie-zrobione-w-1-1


### Info

W tym repozytorium znajdują się kluczowe skrypty mojego projektu, który próbuje odwzorować działanie przeglądarki Google Chrome w Robloxie.
Skrypty, które wykonują mało pracy lub są bardzo proste, nie są tutaj podane, aby zaoszczędzić mój czas i utrzymać projekt czytelny.

### Server

Serwerowe skrypty odpowiadają za:

- Obsługę bazy danych stron (DataStoreService)
- Walidację i przechowywanie nowych stron
- Obsługę wyszukiwania w cache i bazie danych
- Zabezpieczenie przed spamowaniem i botami

Ważne: Logika serwera jest izolowana od klienta, aby nikt nie mógł zmienić wyników ani danych stron.

### Local / Client

Skrypty lokalne odpowiadają za:

- Interfejs użytkownika (UI)
- Obsługę wpisywania adresów i wyszukiwania
- Poprawki literówek i podpowiedzi AI
- Animacje typu typewriter i efektów wizualnych

### Funkcje

- Wyszukiwarka z podpowiedziami
- Obsługa własnych stron i starterowych stron systemowych
- System cooldown i zabezpieczenia przed spamem
- Integracja z modułem AI dla generowania opisów

  ---

### Instalacja / Użycie

aktualnie nie można instalować tego ręcznie w roblox studio, ponieważ ciągle są aktualizacje oraz trzeba przebudować całe UI(a jest tego ogrom)
